A2. Test for Change in Pain Over Time (Primary Hypothesis)  
----------------------------------------------------------  

Given ordinal repeated data (3 time points, same subjects, small N), the **default primary test should be a nonparametric repeated‑measures test**.

**Preferred primary test**

**Test name**  
- Friedman test (nonparametric repeated‑measures ANOVA for k related samples).

**Purpose**  
- To test whether the distribution of NRS scores differs across the three time points (6, 24, 48 hours), accounting for within‑patient pairing.

**Why appropriate (≥2 reasons)**  
1. Repeated measures within the same individuals (3 related samples); Friedman explicitly handles paired ordinal data.  
2. NRS is ordinal and may be non‑normally distributed with potential floor effect at later timepoints; Friedman does not assume normality of differences.  

**Why it may not be appropriate (≥1 reason)**  
- Friedman assumes that observations are at least ordinal and that there are no missing data or that missingness is minimal and completely at random. If many subjects miss a time point, the test loses power or cannot be applied.  

**When this test should NOT be used**  
- When there is substantial missing data at any time point or when the repeated measures structure is irregular (e.g., different number of observations per subject).  
- When group comparisons are needed between independent groups (not the case here).

**Alternative if assumptions are violated**  
- If we have substantial missing data or want to model time continuously:  
  - Use a **linear mixed‑effects model** for NRS treated as continuous (with appropriate checks and robust SEs), or  
  - Use a **generalized linear mixed model (GLMM)** with cumulative logit link for ordinal outcomes.  
- If we desire a simpler, pairwise approach: use **Wilcoxon signed‑rank tests** between pairs (6 vs 24, 24 vs 48, 6 vs 48), with multiplicity adjustment.

A3. Post‑hoc Pairwise Comparisons Between Time Points  
-----------------------------------------------------  

If Friedman test is significant, perform pairwise comparisons to identify where differences lie.

**Test name**  
- Wilcoxon signed‑rank tests (paired, nonparametric) comparing:  
  - 6 vs 24 hours  
  - 24 vs 48 hours  
  - 6 vs 48 hours  
- With Bonferroni or Holm adjustment for multiple comparisons.

**Purpose**  
- To determine at which intervals pain significantly decreases (or changes).

**Why appropriate (≥2 reasons)**  
1. Data are paired and ordinal; Wilcoxon signed‑rank is appropriate for paired non‑normal data.  
2. Small sample (n=22) and possible ceiling/floor effects make nonparametric tests more robust than paired t‑tests.  

**Why it may not be appropriate (≥1 reason)**  
- If many pairs have zero difference (e.g., identical scores at both times), Wilcoxon can lose power or become uninformative.  
- Multiple comparisons increase type I error if not adjusted; with only 22 patients, type II error (false negatives) is also a concern after adjustment.

**When this test should NOT be used**  
- When outcome is highly discrete with limited variability and a very large number of ties, or when the measurement is purely categorical (e.g., only categories none/mild/moderate/severe without the 0–10 scale).  
- When there is substantial missing data in a given pair.

**Alternative if assumptions are violated**  
- For many ties and purely ordered categories: use **Stuart–Maxwell test** for marginal homogeneity or **McNemar‑Bowker test** on categorical pain levels (none/mild/moderate/severe).  
- For more complex modeling: **ordinal logistic mixed models**.

TASK 3 – INAPPROPRIATE TESTS  
=============================  

Below are tests that **should NOT** be used, with reasons.

1. **Independent‑samples t‑test / unpaired tests between time points**  
   - Inappropriate because:  
     - The same patients are measured at different times; data are **paired**, not independent.

2. **One‑way ANOVA for independent groups across time points**  
   - Inappropriate because:  
     - Again, repeated measures on the same subjects; independence assumption violated.  
     - Repeated measures ANOVA or Friedman is required.

3. **Chi‑square test of independence treating time points as independent samples**  
   - Inappropriate because:  
     - Pain categories at different times come from the same subjects; using a standard chi‑square test for independent samples ignores pairing.  
     - Pairwise McNemar or McNemar‑Bowker / Stuart–Maxwell tests for paired categorical data should be used instead.

4. **Pearson correlation between time and pain ignoring repeated measures**  
   - Inappropriate because:  
     - Time is discrete and structured (3 fixed visits), not a continuous random variable; a correlation coefficient would ignore the within‑subject structure.  
     - Longitudinal methods or repeated‑measures tests are more appropriate.

5. **Multiple linear regression with pain at each time point as separate, independent outcomes**  
   - Inappropriate because:  
     - Fails to account for correlation of repeated measures within subjects.  
     - Mixed‑effects models or repeated‑measures ANOVA are required for longitudinal data.

6. **Parametric tests assuming interval scale without checks (e.g., paired t‑tests as primary)**  
   - While sometimes acceptable in practice, using paired t‑tests as the **sole primary analysis** on small‑sample ordinal NRS data without verifying distribution or robustness is not recommended.

7. **Survival analysis (Kaplan–Meier) for “time to no pain”**  
   - Inappropriate here because:
