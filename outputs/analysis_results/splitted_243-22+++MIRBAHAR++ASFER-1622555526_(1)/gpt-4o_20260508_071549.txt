### TASK 2: STATISTICAL ANALYSIS PLAN

#### CATEGORY A – MANDATORY
1. **Test Name**: Chi-Square Test (or Fisher’s Exact Test if expected cell counts are small).
   - **Purpose**: To compare categorical outcomes (e.g., success vs. failure) between Group A and Group B.
   - **Why Appropriate**:
     1. The outcome variable is categorical.
     2. The groups are independent.
   - **Why It May Not Be Appropriate**:
     - If sample sizes are too small for valid chi-square approximation.
   - **When This Test Should NOT Be Used**: If expected frequencies in any cell are <5.
   - **Alternative if Assumptions Are Violated**: Fisher’s Exact Test.

2. **Test Name**: Independent Samples t-Test (or Mann-Whitney U Test if data are non-normal).
   - **Purpose**: To compare continuous variables (e.g., healing time) between Group A and Group B.
   - **Why Appropriate**:
     1. The independent variable (operator type) has two groups.
     2. The dependent variable (e.g., healing time) is continuous.
   - **Why It May Not Be Appropriate**:
     - If the data are not normally distributed or variances are unequal.
   - **When This Test Should NOT Be Used**: If data are heavily skewed or contain outliers.
   - **Alternative if Assumptions Are Violated**: Mann-Whitney U Test.

3. **Test Name**: Kaplan-Meier Survival Analysis (with Log-Rank Test).
   - **Purpose**: To compare the time-to-event data (e.g., time to healing) between the two groups.
   - **Why Appropriate**:
     1. The study involves time-to-event data (e.g., healing time).
     2. The groups are independent.
   - **Why It May Not Be Appropriate**:
     - If follow-up times are inconsistent or censored data are minimal.
   - **When This Test Should NOT Be Used**: If there are no censored observations.
   - **Alternative if Assumptions Are Violated**: Cox Proportional Hazards Model.
