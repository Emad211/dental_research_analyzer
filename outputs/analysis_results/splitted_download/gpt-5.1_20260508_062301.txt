A1. Descriptive statistics and baseline comparability  

a) Test name  
- Descriptive statistics + baseline group comparison  
  - Means (±SD) or medians (IQR) for continuous baseline variables (e.g., age, preoperative VAS).  
  - Frequencies and percentages for categorical variables (e.g., sex, tooth position if recorded).  
  - Between-group comparisons:  
    - Independent samples t-test or Mann–Whitney U test for continuous variables.  
    - Chi-square or Fisher’s exact test for categorical variables.  

b) Purpose  
- Summarize participant characteristics at baseline and verify that randomization produced comparable groups.  
- Describe preoperative pain levels to ensure no systematic group differences.  

c) Why appropriate (≥2 reasons)  
1. Randomized trial: Baseline comparisons are standard to demonstrate balance (even if not strictly necessary).  
2. Age and preoperative VAS may influence pain trajectory; we need to confirm they are comparable, or adjust if imbalances occur.  

d) Why it may not be appropriate (≥1 reason)  
- Overemphasis on statistical significance for baseline differences in randomized trials is discouraged; randomization already ensures balance on expectation.  

e) When not to use  
- Do not formally test baseline differences if sample is very small and tests would be underpowered; instead report descriptive statistics only.  
- Avoid multiple testing on many baseline variables with inferential p-values (risk of spurious imbalances).  

f) Alternative if assumptions violated  
- If continuous variables are skewed or variances unequal, use Mann–Whitney U or robust t-tests (e.g., Welch’s t-test) instead of standard t-test.

A2. Primary analysis: Comparison of VAS pain trajectories over time between groups  

a) Test name  
Option 1 (parametric):  
- Two-way repeated measures ANOVA (mixed design ANOVA) with:  
  - Between-subject factor: Group (NaOCl solution vs gel).  
  - Within-subject factor: Time (6 h, 24 h, 48 h, 72 h, 1 week).  

Option 2 (preferred for robustness):  
- Linear mixed-effects model (LMM) with random intercept (and optionally random slope) for each patient; fixed effects for Group, Time, and Group×Time interaction.  

b) Purpose  
- To test whether postoperative pain (VAS) changes over time and whether that change differs between NaOCl solution and gel groups.  

c) Why appropriate (≥2 reasons)  
1. The design is longitudinal with repeated VAS measures per participant; LMM or repeated-measures ANOVA explicitly models within-subject correlation.  
2. The main scientific question is about group differences in pain over time; these models estimate the Group×Time interaction and overall group effect.  
3. Randomization ensures independence between subjects, and measuring VAS repeatedly aligns with continuous outcome modeling.  

d) Why it may not be appropriate (≥1 reason)  
- VAS scores may be non-normal or heavily skewed, especially early time points with high pain and late time points with many zeros.  
- Some participants are lost to follow-up, leading to missing data which violate the sphericity and complete-case assumptions of repeated-measures ANOVA.  
- VAS is technically ordinal although commonly treated as continuous.  

e) When this test should NOT be used  
- Repeated-measures ANOVA should not be used if:  
  - There is substantial missing data at follow-up times (not MCAR).  
  - The sphericity assumption is badly violated and cannot be adequately corrected (Greenhouse–Geisser etc.), especially with small sample sizes.  
- LMM should not be used if:  
  - The number of participants is too small to estimate random effects reliably (not the case here; n≈100).  

f) Alternative if assumptions are violated  
- If VAS distribution severely departs from normality and is bounded with many zeros:  
  - Use generalized linear mixed models (GLMM) with appropriate link (e.g., Poisson, negative binomial, or beta regression if rescaled 0–1; or a hurdle model if many zeros).  
- If analysis is focused on ordinal categories:  
  - Use cumulative link mixed models (ordinal mixed models).  
- If modeling is not feasible, a nonparametric approach:  
  - Use nonparametric repeated measures tests (e.g., Brunner–Langer rank-based ANOVA or Friedman test within groups, plus between-group comparisons with aligned rank tests).

A3. Between-group comparisons at each time point (secondary)  

a) Test name  
- Independent samples t-test at each time point (if parametric assumptions roughly hold).  
- If not: Mann–Whitney U test for each time point.  

b) Purpose  
- To compare mean (or median) pain intensity between NaOCl gel and solution groups at each follow-up time (6 h, 24 h, 48 h, 72 h, 1 week).  

c) Why appropriate (≥2 reasons)  
1. Provides clinically interpretable comparisons at specific clinically relevant times (e.g., 24 h).  
2. VAS is approximately continuous; groups are independent; sample size ~50 per group gives adequate power.  

d) Why it may not be appropriate (≥1 reason)  
- Multiple testing across time points inflates Type I error unless adjusted.  
- Ignores within-subject correlations and therefore is less efficient and coherent than a longitudinal model.  

e) When this test should NOT be used  
- If a mixed model or repeated measures model already provides the main inference; time-point-specific tests might be redundant or require strong multiplicity control.  
- When sample size at a given time point is very small due to attrition (e.g., <20 per group).  

f) Alternative if assumptions are violated  
- If VAS strongly non-normal:  
  - Use Mann–Whitney U (for continuous but non-normal) or  
  - Use permutation tests to obtain robust p-values.

A4. Analysis of categorical pain levels at each time point  

a) Test name  
- Chi-square test of independence for group vs pain category (no/mild/moderate/severe) at each time point.  
- If expected cell counts <5: Fisher’s exact test.  

b) Purpose  
- To compare distribution of clinically meaningful pain severity categories between NaOCl solution and gel groups over time.  

c) Why appropriate (≥2 reasons)  
1. Pain categories are ordinal but can be analyzed as nominal using chi-square; this is common in clinical pain research.  
2. The clinically relevant question is often “what proportion of patients are pain-free or with mild pain” at each time; chi-square directly tests this.  

d) Why it may not be appropriate (≥1 reason)  
- Does not exploit the ordered nature of categories; loses power compared to ordinal models.  
- Multiple testing across time points and categories increases Type I error.  

e) When this test should NOT be used  
- When many cells have expected counts <5 (especially 0), which invalidates chi-square approximations.  
- When sample size is too small to support a 4×2 contingency table.  

f) Alternative if assumptions are violated  
- Use Fisher’s exact test (2×k).  
- Use ordinal logistic regression (proportional odds model) or ordinal mixed models to handle ordered categories and repeated measures.

A5. Handling missing data  

a) Test name / method  
- Mixed models inherently handle missing at random (MAR) data.  
- If ANOVA is used, define clearly whether analysis is intention-to-treat (ITT) with simple imputation or per-protocol.  

b) Purpose  
- To ensure attrition (loss to follow-up, discontinued due to file separation) does not bias results and that analytic population is clearly defined.  

c) Why appropriate (≥2 reasons)  
1. Longitudinal trials frequently have missing outcome data; ignoring this can bias results.  
2. Mixed models allow inclusion of patients with incomplete data, preserving power and reducing bias under MAR assumptions.  

d) Why it may not be appropriate (≥1 reason)  
- If missingness is not at random (MNAR), mixed models may still be biased.  

e) When this method should NOT be used  
- Do not use single last observation carried forward (LOCF); it biases trajectories in pain studies where pain decreases over time.  

f) Alternative if assumptions are violated  
- Conduct sensitivity analyses using multiple imputation.  
- Perform best/worst-case scenario analyses (for robustness).

TASK 3: INAPPROPRIATE TESTS  

Below are statistical tests or approaches that should NOT be used for this study, with explanations.

1. Multiple independent t-tests without adjustment and ignoring repeated measures  
- Why inappropriate:  
  - Ignoring within-subject correlation violates independence assumptions; leads to incorrect standard errors and p-values.  
  - Conducting separate t-tests at each time point without adjustment inflates Type I error and fragments the primary analysis.  
- Better alternative: LMM or repeated measures ANOVA as described above, with appropriate multiplicity handling for any post-hoc tests.

2. Paired t-tests within groups only  
- Why inappropriate:  
  - Paired t-tests comparing each time to baseline within each group do not address the between-group treatment comparison.  
  - They produce multiple p-values without a coherent summary of treatment effect.  
- Better alternative: Mixed model with Group×Time interaction.

3. Simple pre-post difference analysis only (baseline vs 1 week)  
- Why inappropriate:  
  - Discards rich information at intermediate time points (6, 24, 48, 72 h).  
  - Does not capture potentially different patterns of early vs late pain relief.  
- Better alternative: Longitudinal modeling across all time points.

4. One-way ANOVA comparing groups at a single time combining all time points  
- Why inappropriate:  
  - Treating all VAS values (across time) as independent ignores repeated measures; pseudo-replicates inflate sample size artificially.  
- Better alternative: LMM or repeated measures ANOVA.

5. Using parametric tests (t-tests, ANOVA) with severe violations without checking assumptions  
- Why inappropriate:  
  - Highly skewed VAS distributions or heavy ceiling/floor effects can undermine parametric inference.  
- Better alternative: Use nonparametric tests or transform outcome, or use GLMM.

6. Pearson correlation to “compare” pain trajectories between groups  
- Why inappropriate:  
  - Correlation is not a group comparison and does not test treatment effect; it measures linear association between continuous variables.  
- Better alternative: Group-based longitudinal models.

7. Chi-square tests for trends over time using repeated data as independent observations  
- Why inappropriate:  
  - Repeated pain categories from same patients are not independent; chi-square that ignores this produces biased p-values.  
- Better alternative: Ordinal mixed models or GEE for repeated categorical outcomes.

8. Last Observation Carried Forward (LOCF) as sole method of dealing with missing data  
- Why inappropriate:  
  - LOCF is known to bias results in settings where outcome trends over time (here, decreasing pain).  
- Better alternative: Mixed models with MAR assumption, supplemented by sensitivity analyses.
