2. Primary comparative test for paired binary outcome
-----------------------------------------------------

**Test name:**  
- McNemar’s test for paired binary data (preferred primary test).  
- If data are instead treated as independent canals: chi-square test or Fisher’s exact test (secondary, but conceptually inferior given design).

Given the design (one test and one control canal per tooth), McNemar’s is statistically the correct primary test.

**Purpose:**  
- To compare the proportion of canals with extrusion between test (collagen barrier) and control within matched pairs (same tooth/child).

**Why appropriate (≥2 reasons):**  
1. Outcome is binary (extruded vs not), and canals are paired within tooth – McNemar’s explicitly accounts for this intra-tooth pairing.  
2. Focuses on discordant pairs (test extruded / control not; control extruded / test not), which directly reflects treatment effect under within-tooth randomization.  

**Why it may not be appropriate (≥1 reason):**  
- If pairing is broken (e.g., if some teeth only have one evaluable canal in either test or control group), McNemar’s cannot use unpaired observations and will lose data; with substantial unpairing, it may be inefficient or biased.

**When this test should NOT be used:**  
- When observations are independent and not paired (e.g., if teeth had random allocation of entire tooth to group rather than canal within tooth).  
- When there are no discordant pairs (all pairs have both extruded or both non-extruded), McNemar statistic is undefined.  
- When sample size of discordant pairs is extremely small, p-values may be unreliable (use exact McNemar).

**Alternative if assumptions are violated:**  
- **Exact McNemar test** (if discordant pair counts are small).  
- **Generalized estimating equations (GEE) with logit link** and robust standard errors, clustering by tooth, as a regression-based alternative.  
- If pairing is lost and canals truly independent: **Fisher’s exact test** (for small counts) or **Pearson chi-square test**.

Implementation details:
- Construct 2×2 table at tooth level:
  - Rows: Test canal (E vs NE).
  - Columns: Control canal (E vs NE).
- Apply McNemar’s with continuity correction or exact variant.
- Report:
  - p-value.
  - Discordant pair counts.
  - Optionally, matched-pairs odds ratio with 95% CI.

3. Estimation of effect size for binary outcome
-----------------------------------------------

**Test/Method name:**  
- Estimation of risk difference, risk ratio, and odds ratio (paired or marginal) with 95% confidence intervals.

**Purpose:**  
- Quantify the magnitude and clinical relevance of effect of collagen barrier on extrusion, beyond hypothesis testing.

**Why appropriate (≥2 reasons):**  
1. Effect estimates and confidence intervals provide more interpretable clinical information (how much risk is reduced) than p-values alone.  
2. CONSORT and modern reporting standards emphasize estimation (effect size with CI) as central to trial reporting.  

**Why it may not be appropriate (≥1 reason):**  
- When sample size is small and event rates are low, interval estimates may be very wide and unstable, particularly for odds ratios; exact methods might be required.

**When this should NOT be used:**  
- Should be avoided only if data are too sparse to calculate meaningful CIs (e.g., zero events in both groups).  
- For paired data, do not use naive “independent sample” risk ratio methods that ignore pairing.

**Alternative if assumptions are violated:**  
- Use:
  - **Conditional logistic regression** for matched pairs to obtain matched odds ratio.  
  - **Exact confidence intervals** for proportions and risk differences (e.g., Clopper-Pearson for binomial, or exact McNemar CI).

Specifics:
- Compute (marginal, canal-level) extrusion proportions in each group and:
  - Risk difference (RD = p_test − p_control).  
  - Relative risk (RR = p_test / p_control).  
  - Odds ratio (OR).  
- For paired data, a matched OR from McNemar/conditional logistic regression is preferable.

TASK 3: INAPPROPRIATE TESTS AND WHY
====================================

Below are tests that **should NOT** be used for this study’s primary question, with reasons.

1. Independent-samples t-test / paired t-test
---------------------------------------------

- Outcome is binary (E vs NE), not continuous; t-tests assume continuous, approximately normally distributed data.
- Using t-test on binary outcome (coding 0/1) can in some designs be equivalent to comparing proportions but is non-standard and mis-specifies variance, especially with small sample size.
- When not to use: Any inferential comparison of extrusion rates between groups.

Alternative: McNemar’s test, chi-square, Fisher’s exact, logistic regression.

2. One-way or two-way ANOVA / repeated-measures ANOVA
-----------------------------------------------------

- Same reason: outcome is binary, not continuous, and design is not about multiple continuous measurements.
- Not appropriate for comparing proportions.

Alternative: Logistic regression or generalized linear models for binary outcomes.

3. Pearson correlation or linear regression for main effect
-----------------------------------------------------------

- Correlation assumes continuous variables; here the outcome is binary and treatment is categorical.
- Linear regression with binary outcome violates assumptions and can produce predicted probabilities outside [0,1].

Alternative: Logistic regression, GEE, or conditional logistic regression.

4. Unpaired (independent) chi-square test as primary analysis
-------------------------------------------------------------

- While a chi-square (or Fisher’s exact) test on canals can be computed, treating canals as independent ignores pairing and within-tooth randomization; p-values and confidence intervals may be invalid.
- It does not match the randomization unit (tooth with paired canals).
- It can be used only as sensitivity/exploratory, not as primary.

Alternative: McNemar’s test (primary), exact McNemar.

5. Survival analysis (Kaplan–Meier, Cox regression) for primary outcome
-----------------------------------------------------------------------

- Primary outcome is immediate post-op extrusion, not time-to-event.
- No follow-up times or censoring structures are present for the main endpoint.

Alternative: Binary outcome methods as above.

6. ROC curve analysis
---------------------

- ROC analysis is used for evaluating classification performance of continuous or ordinal predictors against a binary gold standard.
- There is no continuous diagnostic test here; only a treatment versus control comparison.

Alternative: Not needed.

7. Nonparametric tests for continuous data (Mann–Whitney U, Wilcoxon signed-rank, Kruskal–Wallis)
--------------------------------------------------------------------------------------------------

- These are for continuous or ordinal outcomes; not binary.  
- Misapplied if used to compare extrusion frequencies.

Alternative: Appropriate binary outcome tests.
