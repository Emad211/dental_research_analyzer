## CATEGORY A — Mandatory tests/models

### A1) Linear mixed-effects model (LMM) for repeated clinical measures (PD, rCAL, recession, KTW)
- **Purpose:** Compare trajectories and changes over time among groups while accounting for within-tooth repeated measures.
- **Why appropriate (≥2):**
  1. Outcomes measured at **baseline, 3, 6, 12 months** → correlated repeated observations; LMM models this correlation via random effects.
  2. Handles **unequal follow-up/missing visits** under MAR without listwise deletion.
- **Why may not be appropriate (≥1):**
  - If residuals are highly non-normal with strong floor effects (e.g., PD bounded at 0) and transformations fail.
- **When NOT to use:**
  - If only one post-baseline measurement exists (not the case here) or if data are strictly ordinal with few categories.
- **Model specification (recommended):**
  - Fixed effects: time (categorical), obturation, graft, obturation×graft, and interactions with time (especially treatment×time).
  - Random effect: subject/tooth intercept (and possibly random slope for time if supported).
  - Primary estimand: difference in **mean change from baseline** at 12 months (estimated marginal means).
- **Alternative if assumptions violated:**
  - **Robust LMM** (sandwich/Huber-White SE), **transformation** (e.g., log(PD+0.5)), or **generalized linear mixed model (GLMM)** with appropriate distribution (e.g., gamma).

### A2) Ordinal regression (preferred) or nonparametric comparison for CBCTPAI
- **Test/model:** **Ordinal logistic regression** (proportional odds) with baseline adjustment; or **ordinal mixed model** if repeated (baseline and 6 months).
- **Purpose:** Compare CBCTPAI healing severity across groups/time.
- **Why appropriate:**
  1. CBCTPAI is **ordinal**, not continuous; ordinal model respects ordering without assuming equal intervals.
  2. Can adjust for baseline CBCTPAI and estimate odds of being in a better healing category.
- **Why may not be appropriate:**
  - Proportional odds assumption may be violated.
- **When NOT to use:**
  - If CBCTPAI is dichotomized into success/failure exclusively (then logistic regression is better).
- **Alternative if assumptions violated:**
  - **Partial proportional odds model**, **multinomial logistic regression**, or **Wilcoxon/Mann–Whitney/Kruskal–Wallis** on change scores (less ideal).

### A3) Logistic regression for “success rate” (binary) at 6 months and/or 12 months
- **Purpose:** Compare probability of radiographic success between modalities.
- **Why appropriate:**
  1. Outcome defined as **success vs not** → binary.
  2. Allows estimation of **risk/odds differences** and adjustment (e.g., baseline CBCTPAI, defect size).
- **Why may not be appropriate:**
  - If success is extremely common/rare causing separation or unstable estimates.
- **When NOT to use:**
  - If outcome is time-to-event (not defined here) or if repeated success over time is jointly modeled without independence (then GEE/GLMM).
- **Alternative if assumptions violated:**
  - **Firth penalized logistic regression** (for separation), or **log-binomial/Poisson with robust SE** to estimate risk ratios.

## CATEGORY B — Recommended tests/models

### B1) Factorial (2×2) hypothesis testing via contrasts within LMM/GLM framework
- **Purpose:** Quantify **main effects** (MTA vs GP; graft vs no graft) and **interaction**.
- **Why appropriate:**
  1. Trial is naturally a **2×2 factorial**; increases power/clarity.
  2. Reduces multiple testing relative to all pairwise group comparisons.
- **Why may not be appropriate:**
  - If strong qualitative interaction exists and main effects become misleading.
- **When NOT to use:**
  - If randomization or protocol deviates such that factorial structure is broken (e.g., substantial crossovers only in one cell).
- **Alternative if assumptions violated:**
  - Report **simple effects** (pairwise group differences) with multiplicity correction.

### B2) ANCOVA-style baseline adjustment (embedded within models)
- **Purpose:** Improve precision for continuous outcomes at follow-up (e.g., 12-month rCAL) by adjusting for baseline.
- **Why appropriate:**
  1. Baseline clinical status strongly predicts follow-up → improves efficiency.
  2. Protects against chance baseline imbalance even under randomization.
- **Why may not be appropriate:**
  - If baseline is measured with substantial error and induces regression dilution.
- **When NOT to use:**
  - If baseline is post-randomization (not the case).
- **Alternative if assumptions violated:**
  - Use **change-score models** (less efficient typically) or measurement-error methods (rarely needed here).

### B3) Generalized estimating equations (GEE) as sensitivity for repeated outcomes
- **Purpose:** Population-averaged estimates for repeated PD/rCAL or repeated binary success.
- **Why appropriate:**
  1. Robust to misspecification of random-effects distribution.
  2. Provides valid SE with correct working correlation (or robust sandwich).
- **Why may not be appropriate:**
  - Less efficient with small samples and missing data not MCAR; needs adequate clusters.
- **When NOT to use:**
  - If interest is subject-specific inference (then mixed models preferred).
- **Alternative if assumptions violated:**
  - Mixed-effects models with robust SE; MI + GLM.

### B4) Multiplicity control for post-hoc pairwise comparisons
- **Test/procedure:** **Holm correction** (or Tukey for all pairwise if balanced and model-based means).
- **Purpose:** Control family-wise error when comparing multiple groups/time points.
- **Why appropriate:**
  1. Four groups → many possible contrasts.
  2. Prevents inflation of false positives in secondary analyses.
- **Why may not be appropriate:**
  - Overly conservative if many exploratory outcomes.
- **When NOT to use:**
  - For a single pre-specified primary contrast only.
- **Alternative if assumptions violated:**
  - Control **FDR** (Benjamini–Hochberg) for exploratory endpoints.

## TASK 3 — Tests that should NOT be used (and why)

1. **Multiple independent-samples t-tests** across groups/time (e.g., comparing each pair at each visit)
   - Ignores repeated-measures correlation; massive multiplicity; inflated Type I error.

2. **One-way ANOVA on post-treatment values without accounting for baseline**
   - Loses precision; vulnerable to chance baseline imbalance; does not use longitudinal information.

3. **Paired t-tests within each group (baseline vs follow-up) as primary evidence**
   - Answers “did it change?” not “did groups differ?”; cannot attribute differences to treatment.

4. **Chi-square tests for CBCTPAI treated as nominal categories**
   - CBCTPAI is ordinal; chi-square discards ordering and is less powerful/less interpretable.

5. **Simple correlation (Pearson) as substitute for treatment-effect modeling**
   - Does not estimate causal group differences; confounded by time and baseline severity.

## TASK 4 — Summary justification of overall approach

A four-arm randomized trial with repeated periodontal measurements requires **longitudinal modeling** to correctly handle **within-tooth correlation**, baseline differences, and missing follow-up. A **factorial (GP vs MTA) × (graft vs no graft)** framework tested within **mixed-effects models** (continuous outcomes) and **ordinal/logistic regression** (CBCTPAI and success) provides unbiased, efficient estimates of treatment effects, supports clinically interpretable contrasts (main effects and interaction), and limits multiplicity compared with many pairwise tests.
